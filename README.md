# Power BI Analytics Portfolio

This repository documents my learning journey in Power BI, business intelligence, dashboard design, and analytical reporting.

The goal of this portfolio is to practise building dashboards from public datasets while developing practical skills in data preparation, data modelling, DAX, KPI design, and insight communication.

This repository is not focused on one large project. Instead, it contains a growing collection of small-to-medium Power BI dashboard projects that show my progression over time.

## Purpose

The purpose of this repository is to:

- Practise Power BI dashboard development
- Build confidence with Power Query and DAX
- Learn how to turn raw datasets into meaningful business insights
- Improve dashboard layout, interactivity, and visual storytelling
- Create academic and portfolio evidence of my data analytics progression

## Tools and Skills

Main tools and concepts practised in this repository include:

- Power BI Desktop
- Power Query
- DAX measures
- Data cleaning
- Data modelling
- Date tables
- KPI cards
- Interactive filters and slicers
- Dashboard layout and formatting
- Business insight writing
